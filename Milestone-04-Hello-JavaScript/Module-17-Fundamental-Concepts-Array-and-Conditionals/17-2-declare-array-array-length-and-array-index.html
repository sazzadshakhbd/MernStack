Array Elements Position

  • এবার আমরা দেখব, Array -এর মধ্যে যে কয়টা উপাদান আছে তার অবস্থান । এটা দেখতে হলে আমাদের যা করতে হবে তা হলঃ
  • var friendAge = [11, 21, 45, 17, 14, 105, 8]
    console.log(friendAge[0]);
    node array.js
    Result: 11
  • var picnicFee = [5000, 2000, 4000, 120]
    console.log(picnicFee[1]);
    node array.js
    Result: 2000
  • var fruits = ["mango", "banana", "orange"]
    console.log(fruits[3]);
    node array.js
    Result: undefined
  • var vowels = ["a", "e", "i", "o", "u"]
    console.log(vowels[3]);
    node array.js
    Result: o
  • Notes: Array -এর মধ্যে যখন কোন একাধিক উপাদান থাকে তাদের অবস্থান টা শুরু হয় সবসময় ০ দিয়ে।

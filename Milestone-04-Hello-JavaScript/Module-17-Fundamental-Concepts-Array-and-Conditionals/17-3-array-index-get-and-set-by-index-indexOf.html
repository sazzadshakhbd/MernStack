Find Element By Index

  • Notes: কোন একটি উপদান দিয়ে যদি Array -এর মধ্যে তার Position খুজতে হয় তাহলে IndexOf দিয়ে তা বের করতে হবে।
  • var numbers = [21, 42, 63, 84, 25, 96, 17, 68, 39];
    var positionIndex = numbers.indexOf(39);
    console.log(positionIndex);
    node index.js
    Result: 8
  • var numbers = [21, 42, 63, 84, 25, 96, 17, 68, 39];
    var positionIndex = numbers.indexOf(42);
    console.log(positionIndex);
    node index.js
    Result: 1
  • var numbers = [21, 42, 63, 84, 25, 96, 17, 68, 39];
    var positionIndex = numbers.indexOf(110);
    console.log(positionIndex);
    node index.js
    Result: -1
  • Notes: যদি কোন Array of Index -এর Position খুজতে গিয়ে আমরা এমন একটা value দিলাম যা আমাদের var এর ভিতরে নেই তাহলে result -1 দেখাবে।

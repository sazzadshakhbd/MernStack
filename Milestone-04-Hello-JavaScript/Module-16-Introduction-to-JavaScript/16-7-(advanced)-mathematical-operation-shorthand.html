কিভাবে variable -এর মান change করতে হবে?

  1. Create shorthand.js file
  2. var tomatoPrice = 25; [Variable Declaration].
    এখানে আমরা যদি একই variable এর শুধু মান change করি তাহলে var -এ কথাটি লিখতে হবে না বাকি যা আছে তা সবই লিখতে হবে।
    যেমনঃ tomatoPrice = 55; [Variable Value Update]
  3. Add Something to the previous value.
    var price1 = 31;
    var price2 = 10;
  4. Set a new value
    price1 = 34;
  5. Add some value to the existing value
    price1 = price1 + 10;
    console.log(price1);
    node shorthand.js
    Result: 44
  6. price1 += 10 [shorthand +=]
    '+=' means adding some value to the existing value.
  7. var age = 15;
    age = age + 1;
    age++;
    ++ means adding 1 to the existing value.
  1. price2 = price2 - 5

  3. price2 -= 5
  4. -= means deducting some values from the current value of the variable.
  5. var love = 100;
    love = -1;
    love--;
    -- means you are deduction 1 from the existing value

  7. Shorthand '*='
  8. price1 *= 5

  9. price1 = price1 / 5
  10. Shorthand '/='
  11. price1 /= 5

Variable Naming Convention

  • Variable name এর ক্ষেত্রে Rules হলঃ
  • Variable name can not be any keywords. Like:
    false, return etc.
  • Variable name has to be i one word. No space. like:
    var my name = 'sazzad'
  • Variable name can not have quotation. Like:
    var "name" = 'sazzad
  • Variable name can not starts with a number but can ends with a number. like:
    var 99club = 1964; // Wrong
    var club99 = '1976' // Right
  • How to use Long Names?
    1. var user-name = 'sazzad' // Wrong can not use dash
    2. var user_name = 'sazzad' // Right Snack Case
    3. var userHomeAddress = 'Rajbari' // Right Camel Case
    4. var UserHomeAddress = 'Rajbari' // Right Pascal Case
  • Notes:
    1. Variable Name একটি শব্দে হতে হবে
    2. Variable Name কোন সংখ্যা দিয়ে হতে পারবে না
    3. Variable Name একাধিক শব্দ দিয়ে হলে Camelcase Use করতে হবে
    4. Variable Name is case-sensitive. Like:
      var person = 'sazzad';
      var Person = 'sazzad'
    5. Keyword are case-sensitive. Like:
      Var = var
  • এই ভিডিও থেমে আমাদের যেটা শিখতে হবে তা হলঃ var declare করতে হলে single একটা meaningfully word দিয়ে declare করতে হবে। যেমনঃ var price = 10;

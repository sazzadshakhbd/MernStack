Variable Types

  1. String Data Types: var - এর value যদি এক বা একাধিক অক্ষর দিয়ে থাকে তাহলে সেটাতে " " / ' ' - Double / Single Quote বসাতে হবে। Zero/More Characters written inside quotes যেমনঃ
    • var location = "Bandor Bon"
    • var person = "Md. Sazzad Ali Shakh"
    • var seecial = 'Alia Vatt'
  2. Numeric Data Types: Number can be written with or without decimals and without quotes. Like:
    • var human = 5;
  3. Boolean Data Types: It can only take the values true or false and written without quotes. Like:
    • var serious = true;
    • var isFullMarks = true;
    • var isSingle = false;
    • Notes Boolean Algebra নামে একটা জিনিস আছে যেখানে 0,1 বুঝায়। এখানে 0 Negative Value that means false. আর 1 Positive Value that means true.

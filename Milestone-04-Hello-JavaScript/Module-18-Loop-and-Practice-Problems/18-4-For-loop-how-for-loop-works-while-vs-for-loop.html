  3. var penGiven = 0;

  4. initialization: এটি হচ্ছে loop এর জন্য একটি variable কে declare করা. এটা loop শুরু হবার সময় একবার execute হয়।
  5. Condition: প্রতিবার loop execute হবার আগে loop টা আবার execute হবে কি না সেটা check করে।
  6. Final Expression: এর সাহায্যে প্রত্যেক iteration এর পর একবার করে loop variable এর value increase/decrease রে।

For Loop

  1. for(var i = 0; i < 10; i++) {
    console.log(i);
}

  2. for(var i = 1; i <= 10; i++) {
    console.log(i);
}

  3. for(var i = 0; i < 20; i += 2) {
    console.log(i);
}
  4. Notes: Addition Assignment Operator(+2)
    (+=) - এই চিহ্নের ডান পাশের value বাম পাশের variable এর সাথে যোগ করবে এবং বাম পাশের variable এ এই নতুন value assign হয়ে যাবে। যেমনঃ 
  5. for(var i = 1; i <= 20; i += 2) {

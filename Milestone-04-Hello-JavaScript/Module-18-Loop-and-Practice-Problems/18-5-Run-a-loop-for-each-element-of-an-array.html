  3. Traget: Display every element of an array 
    var numbers = [45, 87, 98, 56, 32, 51,25];
for(var i =0; i < 7; i++){
    var number = numbers[i];
    console.log(number)
}

    এই example টি debug করে দেখানো হয়েছে

  4. এই পদ্ধতিতে একটি সমস্যা হচ্ছে আমি যদি আরও উপদান দেই array-তে তাহলে সেটা আর দেখাবে না কারণ আমরা array তে (i < 7) fixed করে দিয়েছি। এই জন্য আমরা (i < 7) এর পরিবর্তে array variable এর নাম দিয়ে তারপর index দিব। যেমনঃ (i < numbers.length) লিখব। তাহলে আমি array এর ভিতরে যদি আরো সংখ্যা বাড়িয়ে দিই তাহলে কোন সমস্যা হবে না।
